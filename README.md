## Install

```bash
# Make it as a dev-only dependency.
yarn add vuepress-plugin-i18n-ui -D
```

## Usage


``` js
// .vuepress/config.js
module.exports = {
  plugins: [ 'i18n-ui' ]
}
```

## Options

### route

- Type: `String`
- Default: `/i18n-ui/`. 

Routing entry of UI.
